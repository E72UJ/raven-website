## 公告
我很高兴地宣布 Raven Engine 0.1.6 正式发布。此版本是一个重要更新，包含许多重要的改进和功能。

Raven Engine 0.1.6 的一些新功能和改进包括：

Raven Engine 现在在所有平台上使用最新的 Rust 1.72 编译器。

资源管理、Vulkan 渲染和物理计算方面的性能均得到了显著提升。

Raven Engine 支持原生加载swf 文件格式。

新的 plugins 和 extensions 目录允许按照优化的加载顺序来有组织地放置插件和扩展。
RVN 配置语言已全面重写，支持嵌套结构、条件编译和模块化导入。
